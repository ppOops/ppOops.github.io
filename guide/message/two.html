API

简要描述：

  • 邮件验证码回执（DR）推送

请求URL：

  • DR推送的Http接口地址（由用户提供）
  • 例如：http://106.15.34.94:8989/email/callback

请求方式：

  • POST

参数：

# Headers：
参数名 参数值 必选 类型 说明
Content-Type application/json 是
# Body：
参数名 类型 必选 说明
messageId string 是 邮件ID
sender string 是 发件人email
receiver array 是 收件人email
price string 是 单价
result string 是 状态: DELIVRD/UNDELIV
sendTime string 是 yyyy-MM-dd HH:mm:ss

返回参数说明

参数名 类型 必选 说明
string 否 success/failure
